What is an IP address?

An IP address (Internet Protocol address) is the number your device uses to send and receive data on a network. Your public IP is the one the rest of the internet sees — assigned by your ISP or hosting provider — and it's what this page detects. There are two versions in use today:

  • IPv4 — four numbers like 203.0.113.24. About 4.3 billion addresses, which the internet long ago outgrew.
  • IPv6 — eight groups of hex like 2606:4700:4700::1111. A vastly larger space built to replace IPv4.

What the summary fields mean

  • ASN — the Autonomous System Number of the network that announces your IP (e.g. AS15169 is Google). It identifies who routes your traffic.
  • Range — the CIDR block your IP belongs to, such as 8.8.8.0/24.
  • Company / Hostname — the organisation operating the network and the reverse-DNS name of the host, if one is set.
  • Privacy & AS Type — whether the IP looks like a VPN, proxy, hosting, or business connection. These are inference features that commercial datasets provide.

How accurate is IP geolocation?

Geolocation maps an IP to a place using registration and routing data — not GPS. Country-level accuracy is very high; city-level is a best guess. Mobile networks, VPNs, and corporate routing can put you hundreds of kilometres from where the data says. Treat the city and coordinates as approximate.

Find your IP from the command line

{% raw %}# Public IP (any OS with curl)
curl https://ipwho.is/ | jq .ip
curl https://ifconfig.me

# Linux / macOS — local interface addresses
ip addr            # Linux
ifconfig           # macOS / BSD

# Windows
ipconfig

# Reverse DNS (hostname) for an IP
dig -x 8.8.8.8 +short
